## How to use it

#### Setting up Poodle
```sh
$ cargo -V
cargo 1.56.0 (4ed5d137b 2021-10-04)
$ cargo install --path .      # install
$ cargo run -- start          # or run directly
```

#### Guess today's riddle
```bash
$ poodle start
```

#### Display full logs
```bash
$ poodle log
```

#### Get help message
```bash
$ poodle --help
```
